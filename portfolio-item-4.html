Savvy SaverBudgeting App

A Comprehensive Financial Management Tool

Description

The SavvySaver Budgeting App is an intuitive and comprehensive tool designed to help users effectively manage their finances. It features secure user registration, budget creation, and transaction logging.

Key Features

    User Account Management
  • Utilization of JSON Web Tokens(JWT) for secure login and user session management.
    Budget Management
  • Users can create a detailed budget, categorized expenses, and monitor their monthly budget. It includes funcaitonalities for real-time tracking of purchases for up to date expense budgeting.
    Interactive User Interface
  • Responsive design with interactive components for a smooth user experience.

Tech Stack

    Front-end
  • ReactJS: Leveraged for building a dynamic and responsive user interface. Utilized React hooks for state management and Context API for global state handling.
  • TailWind CSS: Used TailWind CSS for responsive design and to give a unique look and feel, enhancing user experience.
  • Integration: Interacted with the backend API for data fetching and management.
    Back-end
  • Node.js & Express.js: Created RESTful APIs using Express. Managed server-side logic and database interactions.
  • MongoDB with Mongoose: Utilized MongoDB for database operations with Mongoose for object data modeling, ensuring efficient data storage and retrieval.
  • JWT for Authentication: Implemented JSON Web Tokens for secure user authentication and session management.
  • Bcrypt.js for Security: Used bcrypt.js for hashing and securing user passwords.
    Full-Stack Deployment
  • Static and Web ServicesThe Savvy Saver Budgeting App is deployed on Render.com, a cloud service provider known for its simplicity and efficiency in hosting web applications. The deployment process involved setting up a build environment, configuring the database, and ensuring continuous deployment from the repository. Render.com's seamless integration with Node.js and MongoDB made it an ideal choice for hosting this full-stack application, providing reliable uptime, scalability, and performance.
