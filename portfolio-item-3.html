Blog App

A Comprehensive Blogging Platform

Description

The Blog App is a web application that allows users to create, edit, and view blog posts. It is a complete solution for managing blog content, incorporating both front-end and back-end components.

Key Features

  • Create Post: Users can create new blog posts, adding a title, summary, and content, along with an optional cover image.
  • Edit Post: Existing posts can be edited to update the title, summary, content, and cover image.
  • View Posts: Users can browse a list of blog posts on the homepage, featuring titles, summaries, authors, and creation dates.
  • View Single Post: Full details of individual blog posts can be viewed by clicking on them.
  • User Authentication: The app includes user registration and login functionalities for secure access.

Tech Stack

  • Front-end: React with React Router and Quill for rich text editing.
  • Back-end: Node.js with Express, MongoDB for data storage, and Mongoose for MongoDB object modeling.
  • Authentication: JWT (JSON Web Tokens) for secure user authentication.
